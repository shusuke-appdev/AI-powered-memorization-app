# 🧠 AI 暗記カードアプリ

このアプリは、Google Gemini AIを使用してテキストから暗記カードを自動生成し、忘却曲線に基づいた復習スケジュールを管理できる学習支援ツールです。

## 動作環境

- Windows
- Python 3.8 以上

## 起動方法

フォルダ内の **`run_app.bat`** をダブルクリックしてください。
自動的に必要なライブラリがインストールされ、アプリがブラウザで起動します。

## 手動での起動方法

コマンドプロンプトやPowerShellで以下のコマンドを実行することでも起動できます。

```bash
# 仮想環境の有効化 (もし作成済みであれば)
.\venv\Scripts\activate

# ライブラリのインストール
pip install -r requirements.txt

# アプリの起動
streamlit run app.py
```

## 使い方

1. **APIキーの設定**: 
    - **推奨**: 環境変数 `GEMINI_API_KEY` にキーを設定しておくと、自動で読み込まれます。
    - **手動**: 設定していない場合は、アプリ起動後にサイドバーに入力してください。
2. **カード作成**: 「カードを追加」メニューでテキストを貼り付け、「生成する」ボタンを押します。
3. **復習**: 「復習する」メニューで、その日復習すべきカードが表示されます。

## データの保存場所

作成したカードのデータは、同じフォルダ内の `flashcards.json` に保存されます。このファイルを削除するとデータが消えますのでご注意ください。
