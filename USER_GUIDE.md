## はじめに

### このアプリでできること
- 覚えたいテキストから**穴埋め式の暗記カード**を作成
- **AI**がテキストを分析して、穴埋め箇所を提案
- 科学的な復習スケジュール（**SM-2アルゴリズム**）で効率的に暗記

### 必要なもの
- インターネット接続
- Webブラウザ（Chrome、Safari、Edge など）
- Google アカウント（APIキー取得に必要）

## アカウント作成

### 手順1: アプリにアクセス
ブラウザでアプリのURLを開きます。

### 手順2: 「新規登録」タブをクリック
画面上部に「ログイン」と「新規登録」の2つのタブがあります。
**「新規登録」**をクリックしてください。

### 手順3: ユーザー情報を入力
以下の情報を入力します：

| 項目 | 説明 | 例 |
|------|------|-----|
| ユーザー名 | 2文字以上で自由に決めてください | taro、hanako |
| パスワード | 4文字以上で自由に決めてください | ****（他人に見せないでください） |
| パスワード確認 | 上と同じパスワードをもう一度入力 | **** |

### 手順4: 「登録」ボタンをクリック
入力が完了したら、**「登録」**ボタンをクリックします。

### 手順5: 登録完了
「登録しました！ログインしてください」というメッセージが表示されたら成功です。

## ログイン

### 手順1: 「ログイン」タブをクリック
画面上部の**「ログイン」**タブをクリックします。

### 手順2: ユーザー名とパスワードを入力
登録時に設定したユーザー名とパスワードを入力します。

**30日間**有効です。毎回入力する手間が省けるので、チェックをおすすめします。

### 手順4: 「ログイン」ボタンをクリック
入力が完了したら、**「ログイン」**ボタンをクリックします。

メイン画面（復習画面）が表示されたらログイン成功です。

## APIキーの取得と設定

カードを作成するには、Google の AI（Gemini）を使うための「APIキー」が必要です。
**無料**で取得できます。

### APIキーとは？
AI（人工知能）を使うための「パスワード」のようなものです。
あなた専用のキーで、他の人には教えないでください。

### 手順1: Google AI Studio にアクセス
ブラウザで以下のURLを開きます：
**https://aistudio.google.com/**

### 手順2: Googleアカウントでログイン
お持ちのGoogleアカウント（Gmail）でログインしてください。
アカウントがない場合は作成してください（無料です）。

### 手順3: 「Get API key」をクリック
画面に表示されている**「Get API key」**または**「APIキーを取得」**ボタンをクリックします。

### 手順4: 「Create API key」をクリック
**「Create API key」**または**「新しいAPIキーを作成」**ボタンをクリックします。

### 手順5: APIキーをコピー
表示されたキー（`AIza...`で始まる長い文字列）を**コピー**します。
「コピー」ボタンをクリックするか、文字列を選択して Ctrl+C（Mac: Command+C）を押します。

### 手順6: アプリに戻る
暗記カードアプリの画面に戻ります。

### 手順7: 「⚙️ APIキー設定」を開く
画面に「⚙️ APIキー設定」という項目があります。
クリックして開いてください。

### 手順8: APIキーを貼り付け
入力欄に、先ほどコピーしたAPIキーを**貼り付け**ます。
Ctrl+V（Mac: Command+V）を押すか、右クリックして「貼り付け」を選んでください。

### 手順9: 「APIキーを保存」をクリック
**「APIキーを保存」**ボタンをクリックします。

### 設定完了
「✅ APIキーが設定されています」と表示されたら成功です。

## カードの作成

### 手順1: 「カードを追加」タブをクリック
画面上部のタブから**「📝 カードを追加」**をクリックします。

### 手順2: カテゴリを選択
カードの分類を選びます。プルダウンから選んでください。
- 民法、商法、刑法、憲法、行政法、民事訴訟法、刑事訴訟法、その他

### 手順3: タイトルを入力
カードにタイトルをつけます（オプション）。
同じテーマのカードをまとめて管理するのに便利です。

**例**: 「不法行為」「契約総論」「第1章」など

### 手順4: 覚えたいテキストを入力
テキストエリアに、覚えたい文章を入力またはコピペします。

**例**:
```
民法第709条は不法行為による損害賠償を規定している。
故意または過失によって他人の権利または法律上保護される利益を侵害した者は、これによって生じた損害を賠償する責任を負う。
```

### 手順5: 「📝 テキストを解析」をクリック
入力が完了したら、**「📝 テキストを解析」**ボタンをクリックします。

### 手順6: AIが文節に分割
AIがテキストを意味のある単位（文節）に分割します。
各文節がボタンとして表示されます。

### 手順7: 穴埋め箇所を選択
穴埋めにしたい箇所（覚えたい用語）を**クリックして選択**します。

- **灰色**: 未選択（そのまま表示される）
- **緑色**: 選択済み（穴埋め「______」になる）

**ポイント**: 専門用語、固有名詞、数字など、覚えるべき重要な語句を選びましょう。

### 手順8: 「🤖 AIに提案させる」（オプション）
どこを穴埋めにすべきか迷ったら、**「🤖 AIに提案させる」**ボタンをクリック。
AIがおすすめの箇所を自動で選んでくれます。
（気に入らなければ、クリックして変更できます）

### 手順9: プレビューを確認
選択した箇所が「______」になったプレビューが表示されます。
問題なければ次へ進みます。

### 手順10: 「✨ カード生成」をクリック
**「✨ カード生成」**ボタンをクリックします。

### 手順11: カードをプレビュー
生成されたカードが表示されます。
- **問題**: 穴埋め部分が「______」になった文章
- **答え**: 穴埋め部分に入る語句

内容を確認し、必要なら編集できます。

### 手順12: 「💾 デッキに保存」をクリック
内容がOKなら、**「💾 デッキに保存」**ボタンをクリックします。

### 作成完了
「○枚のカードを保存しました」と表示されたら完了です！

## カードで復習する

### 手順1: 「復習する」タブをクリック
画面上部のタブから**「📚 復習する」**をクリックします。

### 手順2: 問題を読む
カードに問題が表示されます。
「______」の部分に何が入るか、頭の中で考えてください。

### 手順3: 「答えを見る」をクリック
答えを考えたら、**「答えを見る」**ボタンをクリックします。

### 手順4: 答えを確認
正解が表示されます。自分の答えと比べてください。

### 手順5: 覚えていた度合いを評価
4つのボタンから、どれくらい覚えていたかを選んでクリックします。

| ボタン | いつ使う？ | 次の復習は？ |
|--------|----------|------------|
| **忘れた (0)** | 完全に忘れていた、答えが全く思い出せなかった | 翌日 |
| **難しい (3)** | かなり苦労して思い出した | 数日後 |
| **普通 (4)** | 少し考えて思い出せた | 約1週間後 |
| **簡単 (5)** | すぐに答えが思い浮かんだ | 2週間以上後 |

**ポイント**: 正直に評価することが大切です。
「簡単」を選びたくなりますが、正直に評価すると、忘れかけた頃に復習できます。

### 手順6: 次のカードへ
評価をクリックすると、自動で次のカードが表示されます。
すべてのカードを復習し終えると「🎉 復習完了！」と表示されます。

### 毎日続けましょう
このアプリは**毎日少しずつ復習する**ことで効果を発揮します。
1日5〜10分でも継続することが大切です。

## カードを管理する

### 手順1: 「カード管理」タブをクリック
画面上部のタブから**「🗂️ カード管理」**をクリックします。

### カテゴリで絞り込み
タブをクリックして、特定のカテゴリのカードだけを表示できます。

### 検索
🔍 検索ボックスにキーワードを入力して、カードを検索できます。
問題文、答え、タイトルが検索対象です。

### カードを編集
各カードの「問題」「答え」「カテゴリ」を編集できます。
変更したら「更新」ボタンをクリックして保存します。

### カードを削除
不要なカードは「🗑️ このカードを削除」ボタンで削除できます。
**確認画面**が表示されるので、間違って削除する心配はありません。

### タイトルグループで一括削除
同じタイトルのカードをまとめて削除したい場合は、
タイトルグループの「🗑️ 全削除」ボタンを使います。

## よくある質問

### Q: データは保存されますか？
**A**: はい。クラウドに保存されるので、ブラウザを閉じてもデータは消えません。
別のパソコンやスマホからも同じアカウントでログインすればアクセスできます。

### Q: 他の人に自分のカードを見られますか？
**A**: いいえ。各ユーザーのデータは独立しており、他の人からは見えません。

### Q: オフラインで使えますか？
**A**: いいえ。このアプリはインターネット接続が必要です。

### Q: APIの無料枠を超えたと表示されました
**A**: Google Gemini API の無料枠には制限があります。
以下の対処法をお試しください：
1. しばらく（数分〜数時間）待ってから再試行
2. 新しいAPIキーを取得して設定

### Q: パスワードを忘れました
**A**: 現在、パスワードリセット機能はありません。
新しいアカウントを作成してください。

### Q: ヘルプはどこにありますか？
**A**: 画面右上の**「❓ ヘルプ」**ボタンをクリックすると、
使い方の概要が表示されます。
